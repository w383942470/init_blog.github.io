Python爬虫遇到中文乱码怎么办

2021-09-08 日常问题 45 字
python

在爬虫过程中遇到页面中文编码乱码的问题，直接给返回值更改编码即可解决

import requests

res = requests.get(url)

res.encoding = 'gb2312'

print(res.text)
